## Description

Aquifer Server is the back-end server allowing access to data in the Aquifer.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Lint

```bash
# lint
$ yarn run lint
```

## Test

```bash
# unit tests
$ yarn run test

# integration tests
$ yarn run test:integration
```
